SUMMARY
-------

DC 2 MODS

This is a custom module that works with Islandora to automatically transform DC datastream to MODS and attach the MODS datastream to the object

REQUIREMENTS
------------

The following Drupal modules are required:
 * islandora
 * zip_importer (A sub module found in the islandora_importer module)

INSTALLATION
------------

Enable the module.
